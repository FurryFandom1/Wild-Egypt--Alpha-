Wild Egypt это онлайн шутер, действия которого происходят в части дикой пустыни, задя на территорию которой, вы пробуждаете злых духов пустыни. Те в ответ воскрешают своих подчинённых да бы защитить свои ценности. Отправляйтесь в бой в одиночку или покоряйте края пустыни в друзьями!

Продукт является демонстрационной игрой с целю демонстрации навыков программирование на языке C# и умения работать с Unity движком. Вся информация перечисленная в данном репозитории написана и создана исключительно человеческими руками и живыми людьми. Нейросети в ходе разработки НЕ ИСПОЛЬЗОВАЛИСЬ

Игра на стадии Alpha
Для перехода на Beta должны быть выполнены следующие пункты:

- [] Играбельная механика перемещения и передвижения персонажа
- [] Умный AI противников
- [] Несколько видов противников
- [] Пару видов карт
- [] Разные виды оружия, которые открываются раз в несколько волн и выдаются в зависимости от колличества убийств
- [] соместимость в онлайн на основе Photon
- [] Понятный и функциональный интерфейс
- [] Список доступных серверов для мультиплеера
- [] заглушка модели игрока



Программы использованные в разработке: Unity 6, Blender 3D, MagicVoxel, Visual Studio Code, GithubDesktop
